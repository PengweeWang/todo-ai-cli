todo 待办工具使用文档

1. 简介

命令行待办管理工具，支持添加、查看、修改、归档待办，及LLM自动提取待办信息。

2. 基本格式

todo [命令] [选项]

命令：add、print、set、archive、config、help

3. 命令说明

3.1 添加待办（add）

todo add [标题] [选项]

选项：

选项说明
-a, --autoLLM自动提取（需配置API）
-d, --deadline截止时间（格式："月.日-时:分"）
-t, --tags标签（如"学习"）
-des, --description描述信息

示例：

todo add "完成作业" -t "学习"
todo add "提交报告" -d "10.25-18:00" -des "季度总结"
todo add "周五3点开会" --auto

3.2 查看待办（print）

todo print [id]

id为序号（留空查所有）

todo print
todo print 3

3.3 修改待办（set）

todo set [id] [选项]

选项：

选项说明
-f, --finish标记完成
-t, --title修改标题
-d, --deadline修改截止时间
-de, --description修改描述
-l, --location修改地点
-ta, --tags修改标签
-r, --remove删除待办
todo set 2 -f
todo set 1 -d "10.26-20:00"
todo set 3 -r

3.4 归档待办（archive）

todo archive [id]

id留空则归档所有已完成项

todo archive
todo archive 5

3.5 配置（config）

todo config [选项]

选项：

选项说明
-k, --key设置API Key
-m, --model设置模型
-a, --api_url设置API地址
todo config -k "your_key"
todo config -m "gpt-3.5-turbo"
todo config # 查看配置

3.6 帮助（help）

todo help

4. 数据存储

  • 待办：~/.local/share/td/todo.json
  • 归档：~/.local/share/td/archive.json
  • 配置：~/.local/share/td/config.ini

5. 状态标识

标识含义
⏩ 蓝色进行中
✅ 绿色已完成
❌ 红色已超时
